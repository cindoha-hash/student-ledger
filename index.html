About

Student Ledger is a finance tracker built for the Building Responsive UI summative assignment. It follows a student-ledger design with a deep navy palette, warm amber accents, and a torn-ledger-edge motif.

Regex Validation Rules

  • /^\S(?:.*\S)?$/ — description
  • /^(0|[1-9]\d*)(\.\d{1,2})?$/ — amount
  • /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/ — date
  • /^[A-Za-z]+(?:[ -][A-Za-z]+)*$/ — category
  • /(?=.*\d)(?=.*[a-zA-Z])(?=.*[^\w\s])/ — advanced

  • Escape — cancel
